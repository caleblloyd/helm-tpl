# tests

## Test a Patch File

```bash
go run . <path-to-doc.json> <path-to-patch.json>
```

## Run All Tests

```bash
go test .
```

The test fixtures in the files `tests.json` and `spec_tests.json` have been copied from [json-patch/json-patch-tests](https://github.com/json-patch/json-patch-tests)
